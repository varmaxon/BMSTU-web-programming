РК №1. Варняга М.О. ИУ6-33Б. Билет №47

Задание:

Создать HTML-страницу, содержащую форму для ввода необходимых данных. Заголовок страницы должен содержать ФИО, группу, вариант и полный текст задания. Реализовать задачу, соответствующую выданному билету, на Javascript и проверить результат. В форме ввода значений должны быть подставлены некоторые значения по умолчанию, позволяющие проверить правильность работы программы. При вводе недопустимых значений должно выдаваться корректное сообщение, из которого должно быть однозначно понятно, что следует исправить во вводимом значении. Для проверки результата выводить исходные и модифицированные значения на экран. Если модификации нет, выдать соответствующее сообщение. Результат загрузить на сайт в виде HTML-страницы.
